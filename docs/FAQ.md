您可以直接在浏览器中输入 https://aiyouxiindex.com.cn 进入爱游戏主页。网站无需注册即可浏览公开内容，部分功能（如评论、收藏）可能需要创建账户。建议使用最新版本的 Chrome、Firefox 或 Edge 浏览器以获得最佳体验。
